# My HTML Game

## Overview
This project is a simple HTML-based game that features a dynamic user interface and engaging gameplay mechanics. The game includes various elements such as a health bar, ammo box, skill dock, and a game menu, all styled to provide an appealing visual experience.

## Project Structure
```
my-html-game
├── src
│   ├── index.html       # Main HTML structure of the game
│   ├── styles.css       # CSS styles for the game
│   └── script.js        # JavaScript code implementing game logic
├── .vscode
│   └── extensions.json   # Recommended extensions for development
└── README.md            # Documentation for the project
```

## Setup Instructions
To set up the project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd my-html-game
   ```

2. **Open the project in your code editor**.

3. **Install recommended extensions**:
   To enhance your development experience, consider installing the following extensions in Visual Studio Code:
   - **Live Server**: For live reloading of HTML files during development.
   - **Prettier**: For code formatting to maintain consistent style across HTML, CSS, and JavaScript files.
   - **ESLint**: For linting JavaScript code to catch errors and enforce coding standards.
   - **HTML CSS Support**: For enhanced HTML and CSS IntelliSense.

4. **Run the game**:
   Open `src/index.html` in your browser or use the Live Server extension to view the game in action.
